few-shot classification

few-shot classification is a task to train a classifier which can classify example among classes not seen in training with only few labeled examples of each class.

N-way K-shot classification

  • input: S1,⋯ ,SN,QS^1, \cdots, S^N, QS1,⋯,SN,Q
    • Sn={s1n,⋯ ,sKn},n∈{1,⋯ ,N}S^n = \{ \mathbb{s}^n_1, \cdots, \mathbb{s}^n_K \}, n \in \{1, \cdots, N\}Sn={s1n​,⋯,sKn​},n∈{1,⋯,N}: class nnn support set (examples with class nnn)
    • Q={q1,⋯ ,qM}Q = \{ \mathbb{q}_1, \cdots, \mathbb{q}_M \}Q={q1​,⋯,qM​}: query set (examples to predict class)
  • output: PPP
    • P={p1,⋯ ,pM}P = \{ \mathbb{p}_1, \cdots, \mathbb{p}_M \}P={p1​,⋯,pM​}: predicted class probabilities

model

structure

  • fθ(⋅)f_\theta(\cdot)fθ​(⋅): embedding function
  • D(⋅,⋅)D(\cdot, \cdot)D(⋅,⋅): distance function

prediction

  1. for nnn in {1,⋯ ,N}\{ 1, \cdots, N \}{1,⋯,N} do
    1. cn←1N∑s∈Snfθ(s)\mathbb{c}^n \leftarrow \frac{1}{N} \sum\limits_{\mathbb{s} \in S^n} f_\theta(\mathbb{s})cn←N1​s∈Sn∑​fθ​(s)
  2. for mmm in {1,⋯ ,M}\{ 1, \cdots, M \}{1,⋯,M} do
    1. em←fθ(qm)\mathbb{e}_m \leftarrow f_\theta(\mathbb{q}_m)em​←fθ​(qm​)
    2. dm←(D(em,c1),⋯ ,D(em,cN))\mathbb{d}_m \leftarrow \left( D(\mathbb{e}_m, \mathbb{c}^1), \cdots, D(\mathbb{e}_m, \mathbb{c}^N) \right)dm​←(D(em​,c1),⋯,D(em​,cN))
    3. pm←softmin(dm)\mathbb{p}_m \leftarrow \text{softmin}(\mathbb{d}_m)pm​←softmin(dm​)

train

  • NCN_CNC​: # way
  • NSN_SNS​: # shot
  • NQN_QNQ​: # query per way
  • loss function: negative log-probability of true class
  • train set: T1,⋯ ,TUT^1, \cdots, T^UT1,⋯,TU
    • Tu={x1u,⋯ ,xVu},u∈{1,⋯ ,U}T^u = \{ \mathbb{x}^u_1, \cdots, \mathbb{x}^u_V \}, u \in \{ 1, \cdots, U \}Tu={x1u​,⋯,xVu​},u∈{1,⋯,U}: VVV examples with class uuu

train is done by iterating over episodes.
